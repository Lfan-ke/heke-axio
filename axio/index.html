Crate axio

Source
Expand description

std::io-like I/O traits for no_std environment.

§Examples

Basic usage with byte slices:

use axio::{Read, BufReader};

let data = b"hello world";
let mut reader = BufReader::new(&data[..]);
let mut buf = [0u8; 5];
reader.read_exact(&mut buf).unwrap();

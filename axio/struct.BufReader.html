Struct BufReader

pub struct BufReader<R> { /* private fields */ }
Expand description

A buffered reader that reads from another reader Re-export the BufReader type from the bufreader module. The BufReader<R> struct adds buffering to any reader.

Source§

fn read_exact(&mut self, buf: &mut [u8]) -> Result<()>

Reads exactly enough bytes to fill the buffer, using buffered data first Small read_exacts from a BufReader are extremely common when used with a deserializer. The default implementation calls read in a loop, which results in surprisingly poor code generation for the common path where the buffer has enough bytes to fill the passed-in buffer.
